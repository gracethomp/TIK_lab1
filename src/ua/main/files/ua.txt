Java — це мова програмування та обчислювальна платформа, яка була вперше випущена компанією Sun Microsystems у 1995 році. Технологія еволюціонувала від скромної розробки до інструменту, який відіграє важливу роль у сучасному цифровому світі, забезпечуючи надійну платформу для багатьох послуг і програм. Інноваційні продукти та цифрові послуги, розроблені для майбутнього, також базуватимуться на Java.

Незважаючи на те, що більшість сучасних програм Java поєднує середовище Java та програму, все ще є багато програм і навіть деякі веб-сайти, які не працюватимуть, якщо на комп’ютері не встановлено версію Java для настільного комп’ютера.

Так, Java можна завантажити безкоштовно для особистого використання.
Java також безкоштовна для розробки: розробники можуть знайти всі набори для розробки та інші корисні інструменти на веб-сторінці. Останні оновлення Java містять важливі вдосконалення для підвищення продуктивності, стабільності та безпеки програм Java, які працюють на вашому комп’ютері. Встановлення цих оновлень гарантує, що програми Java продовжуватимуть працювати з останньою версією. У результаті завантаження програмного забезпечення Java з java.com користувачі отримують Java Runtime Environment (JRE) версії 8.

JRE складається з віртуальної машини Java (JVM), базових класів платформи Java та бібліотек підтримки платформи Java. JRE — це середовище виконання для Java. Це все, що потрібно для запуску програм Java WebStart із підтримуваного веб-браузера. Однак інструменти розробки не включені: вони надаються як частина Java Development Kit (JDK).
Що таке плагін Java?
Програмний модуль Java є компонентом Java Runtime Environment (JRE). JRE дозволяє деяким програмам, написаним на мові програмування Java, працювати в різних браузерах. Плагін Java не є окремою програмою і не може бути встановлений окремо.
Я зустрічав терміни «Віртуальна машина Java» та JVM. Це програмне забезпечення Java?
Віртуальна машина Java — це лише один аспект програмного забезпечення Java, який бере участь у запуску програми. Віртуальна машина Java, як частина JRE, вбудована в програмне забезпечення Java, яке можна завантажити; він підтримує запуск програм Java.
термінологія. Java — це загальний термін, який використовується для позначення програмного забезпечення та його компонентів, включаючи компоненти Java Runtime Environment (JRE) і Java Virtual Machine (JVM), а також плагін. Якщо повідомлення про помилки містять терміни JRE, JVM і плагін, ми зберігаємо їх.

Версії: версія відноситься до сімейства Java і номера оновлення. Приклади: на веб-сайті або в програмах Windows версія вказана як Java 8 Update 25. Більш ранні версії також можуть мати позначення 1.7.0_65, що означає Java 7 (оновлення 65).
Розділ довідки містить інформацію про такі версії Java. Лямбда-вирази були потужним доповненням до мови Java, починаючи з Java 8. Це серія навчальних посібників, спрямованих на ознайомлення з концепцією лямбда-виразів і поступове навчання їх використання на практиці в міру проходження кожного посібника.

Підручники з цієї серії наведено нижче. Ми рекомендуємо починати з першого та розвиватися далі, але ви можете почати з будь-якого місця! У 2014 році в Java SE 8 було введено концепцію лямбда-виразів. Якщо ви пам’ятаєте дні до випуску Java SE 8, то, напевно, пам’ятаєте концепцію анонімних класів. І, можливо, ви чули, що лямбда-вирази є іншим, простішим способом написання екземплярів анонімних класів, у деяких точних випадках.

Якщо ви не пам’ятаєте тих часів, то, можливо, ви чули або читали про анонімні класи і, ймовірно, боїтеся цього незрозумілого синтаксису.

Що ж, хороша новина: вам не потрібно проходити анонімні заняття, щоб зрозуміти, як написати лямбда-вираз. Крім того, у багатьох випадках завдяки додаванню лямбда-виразок до мови Java вам більше не потрібні анонімні класи.
Усе має тип у мові Java, і цей тип відомий під час компіляції. Тому завжди можна знайти тип лямбда-виразу. Це може бути тип змінної, поля, параметра методу або повернутий тип методу.
